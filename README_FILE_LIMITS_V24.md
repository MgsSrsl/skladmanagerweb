# v24 — лимиты файлов и сжатие фото

Заменить/добавить файлы из архива в корень проекта.

Изменено:
- `act-client.html` — клиентская форма: до 10 файлов на акт, один файл до 10 МБ, фото автоматически сжимаются перед загрузкой.
- `acts/akt-create.html` — загрузка готового акта: один файл до 10 МБ, фото сжимаются.
- `acts/akt-journal.html` — тот же upload helper для совместимости.
- `acts/akt-admin.html` — копия журнала, тоже обновлена.

Что считается лимитом 10 файлов:
- уже приложенные файлы + новые выбранные файлы в клиентской форме.
- если старых файлов уже 10, добавить новые нельзя.

Сжатие фото:
- максимальная сторона: 1800 px;
- JPEG качество: 0.82;
- если после сжатия файл всё равно больше 10 МБ, загрузка остановится с ошибкой.
