# Акты Docke — PWA v19

Файлы вынесены в отдельную папку `/acts/`.

Страницы после загрузки на Vercel/GitHub:

- `/acts/` — стартовая страница приложения
- `/acts/akt-create.html` — создание ссылки клиенту / загрузка готового акта
- `/acts/akt-journal.html` — журнал актов
- `/acts/akt-settings.html` — админка
- `/acts/akt-client.html?id=...` — форма клиента

Приложение можно установить как PWA из Chrome/Edge через кнопку «Установить» или меню браузера.


## v21

Кнопка «Установить приложение» теперь показывается всегда на рабочих страницах. Если браузер не отдаёт системное окно установки, кнопка показывает понятную инструкцию для Chrome/Edge/Android/iPhone.


## Проверка установки
Откройте `/acts/pwa-check.html`, чтобы проверить HTTPS, manifest и service worker. В Edge установка может быть спрятана: меню `...` → `Другие инструменты` → `Приложения` → `Установить этот сайт как приложение`. Если Edge показывает «Управляет ваша организация», установка PWA может быть отключена политикой браузера.
